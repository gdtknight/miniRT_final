## 🔄 GitHub Wiki 동기화

이 디렉토리의 마크다운 파일을 GitHub Wiki에 수동으로 업로드해야 합니다.

### 방법 1: 웹 인터페이스

1. GitHub 저장소의 Wiki 탭으로 이동
2. "New Page" 클릭
3. 파일 이름과 내용 복사/붙여넣기
4. "Save Page" 클릭

### 방법 2: Git Clone (권장)

```bash
# Wiki 저장소 클론
git clone https://github.com/YOUR_USERNAME/miniRT.wiki.git

# Wiki 파일 복사
cp wiki/*.md miniRT.wiki/

# 커밋 및 푸시
cd miniRT.wiki
git add .
git commit -m "Update wiki from local source"
git push
```

## ✏️ 문서 작성 규칙

### 파일 명명

- **한글 제목**: `프로그램-개요.md`
- **영문 제목**: `Quick-Start-Guide.md`
- **공백 대신 하이픈** 사용

### 마크다운 스타일

- **헤더**: `#`, `##`, `###` (계층 구조)
- **코드 블록**: ` ```language ` (언어 지정)
- **링크**: `[텍스트](파일명.md)` (상대 경로)
- **강조**: `**굵게**`, `*기울임*`, `` `코드` ``

### 구조

모든 문서는 다음 구조를 따릅니다:

```markdown
# 문서 제목

간단한 설명

---

## 섹션 1

내용...

---

## 섹션 2

내용...

---

## 🔗 관련 문서

- [다른 문서](다른-문서.md)

---

**Last Updated**: YYYY-MM-DD  
**Version**: X.X.X
```
