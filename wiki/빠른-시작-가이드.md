# 빠른 시작 가이드

이 가이드는 miniRT를 처음 시작하는 분들을 위한 단계별 설치 및 실행 방법을 제공합니다.

---

## 📋 필수 요구사항

### 운영 체제
- **Linux**: Ubuntu 20.04 이상, Debian, Fedora 등
- **macOS**: 10.14 (Mojave) 이상

### 개발 도구
- **컴파일러**: GCC 7.0+ 또는 Clang 6.0+
- **빌드 도구**: GNU Make 3.81+
- **Git**: 버전 관리 (선택사항)

### 시스템 라이브러리

#### Linux (Ubuntu/Debian)
```bash
sudo apt-get update
sudo apt-get install -y \
    libx11-dev \
    libxext-dev \
    libbsd-dev \
    gcc \
    make
```

#### macOS
```bash
# Xcode Command Line Tools 설치
xcode-select --install
```

## 🚀 설치

### 1. 저장소 클론

```bash
git clone https://github.com/YOUR_USERNAME/miniRT.git
cd miniRT
```

### 2. 빌드

```bash
# 프로젝트 빌드
make

# 성공 메시지 확인
# "miniRT compiled successfully" 또는 유사한 메시지
```

#### 빌드 옵션
```bash
make              # 일반 빌드
make re           # 재빌드 (clean + all)
make clean        # 오브젝트 파일 삭제
make fclean       # 모든 빌드 파일 삭제
```

### 기본 실행

```bash
./miniRT scenes/test_simple.rt
```

### 예제 장면들

#### 1. 간단한 테스트
```bash

```
- 1개의 구체와 평면
- 기본 조명 설정
- 초보자에게 적합

#### 2. 조명 효과
```bash
./miniRT scenes/test_lighting.rt
```
- 다양한 조명 효과
- Phong 조명 모델 시연
- Specular highlights

#### 3. 원기둥 테스트
```bash

```
- 여러 원기둥 배치
- 다양한 각도 및 크기

#### 4. 종합 테스트
```bash

```
- 40개 이상의 객체
- 복잡한 장면 구성
- BVH 최적화 효과 확인

## 🎮 기본 조작법

프로그램이 실행되면 다음 키로 조작할 수 있습니다:

### 카메라 이동
- **W** - 앞으로 이동
- **X** - 뒤로 이동
- **A** - 왼쪽으로 이동
- **D** - 오른쪽으로 이동
- **Q** - 위로 이동
- **Z** - 아래로 이동

### 카메라 회전
- **E** - 위로 회전 (피치)
- **C** - 아래로 회전 (피치)
- **S** - 초기 위치로 리셋

### UI 컨트롤
- **H** - HUD 표시/숨김
- **I** - 성능 정보 표시/숨김
- **ESC** - 프로그램 종료

자세한 조작법은 [인터랙티브 컨트롤](인터랙티브-컨트롤.md)을 참조하세요.

## 🎨 첫 장면 만들기

### 1. 장면 파일 생성

`my_scene.rt` 파일을 생성하고 다음 내용을 입력합니다:

```rt
# 주변광
A  0.2  255,255,255

# 카메라
C  0,0,20  0,0,-1  70

# 조명
L  -10,10,10  0.8  255,255,255

# 빨간 구체
sp  0,0,0  10  255,0,0

# 회색 평면 (바닥)
pl  0,-5,0  0,1,0  200,200,200
```

### 2. 실행

```bash
./miniRT my_scene.rt
```

### 3. 장면 수정

- 구체 색상 변경: `sp  0,0,0  10  0,255,0` (녹색)
- 구체 크기 변경: `sp  0,0,0  20  255,0,0` (지름 20)
- 구체 위치 변경: `sp  5,0,0  10  255,0,0` (오른쪽으로 5만큼)

자세한 파일 형식은 [씬 파일 형식](씬-파일-형식.md)을 참조하세요.

## ✅ 설치 검증

### 1. 빌드 확인

```bash
# 실행 파일 존재 확인
ls -lh miniRT

# 예상 출력: -rwxr-xr-x ... miniRT
```

### 2. 기본 실행 테스트

```bash
# 간단한 장면 실행
./miniRT scenes/test_simple.rt

# 창이 열리고 렌더링된 이미지가 표시되어야 함
```

### 3. 메모리 검사 (선택)

#### Linux
```bash
valgrind --leak-check=full ./miniRT scenes/test_simple.rt
# 메모리 누수가 없어야 함
```

#### macOS
```bash
leaks -atExit -- ./miniRT scenes/test_simple.rt
# "0 leaks for 0 total leaked bytes" 확인
```

## 🐛 문제 해결

### 빌드 오류

#### "mlx.h: No such file or directory"
```bash
# MinilibX 서브모듈 업데이트
git submodule update --init --recursive
make re
```

#### "X11/Xlib.h: No such file or directory" (Linux)
```bash
# X11 개발 라이브러리 설치
sudo apt-get install libx11-dev libxext-dev
```

### 실행 오류

#### "Error: Failed to open file"
```bash
# 파일 경로 확인
ls -la scenes/test_simple.rt

# 상대 경로 대신 절대 경로 사용
./miniRT /full/path/to/scene.rt
```

#### "Error: Invalid scene format"
```bash
# 파일 형식 검증
cat scenes/test_simple.rt

# UTF-8 인코딩 확인
file scenes/test_simple.rt
```

### 화면 표시 오류

#### "Error: mlx_init failed" (Linux)
```bash
# X11 서버 실행 확인
echo $DISPLAY

# DISPLAY 환경 변수 설정
export DISPLAY=:0
```

#### 창이 열리지 않음 (macOS)
```bash
# 접근 권한 확인
# 시스템 환경설정 > 보안 및 개인정보 보호 > 접근성
```

더 많은 문제 해결 방법은 [문제 해결](문제-해결.md) 페이지를 참조하세요.
