## [2.1.0] - 2026-01-15

### 주요 변경사항

- **BVH 가속 구조 (Spec 017)**: 복잡한 장면에서 2-10배 렌더링 속도 향상
  - 기본 활성화로 별도 설정 불필요
  - O(log n) 트리 순회로 교차 검사 최적화
  - 50개 이상 객체 장면에서 최대 성능 발휘

- **디바운싱 렌더링 (Spec 018)**: 90% 이상 불필요한 렌더링 제거
  - 150ms 디바운스 타이머로 입력 버퍼링 제거
  - 4-상태 기계 (IDLE → ACTIVE → PREVIEW → FINAL)
  - 부드러운 인터랙션과 즉각적인 시각 피드백 제공

- **프리뷰 모드**: 인터랙션 중 즉각적인 시각적 피드백
  - 저품질 프리뷰 렌더링으로 즉시 반응
  - 자동으로 고품질 최종 렌더링으로 전환

- **100% Norminette 준수 (Spec 016)**: 모든 코드가 42 School 코딩 규칙 준수
  - 전체 코드베이스 norminette 통과
  - 일관된 코드 스타일과 품질

- **통합 객체 식별자 (Spec 015)**: 일관된 객체 관리 시스템

- **BVH 트리 시각화 (Spec 014)**: `--bvh-vis` 플래그로 디버그 시각화 지원

### 성능 개선

- BVH 가속: 복잡한 장면(50+ 객체)에서 2-10배 속도 향상
- 디바운싱: 인터랙션 중 불필요한 렌더링 90% 이상 감소
- 메모리 누수 제로, 새 코드에서 힙 할당 제로

### 버그 수정

- BVH 활성화 시 검은 화면 문제 해결
- 렌더링 중일 때만 취소하도록 로직 개선
- 사양 번호 일관성 수정

### 문서화

- 18개 기능 사양 문서 작성
- 포괄적인 Doxygen 스타일 주석
- API 계약 및 데이터 모델 완성
