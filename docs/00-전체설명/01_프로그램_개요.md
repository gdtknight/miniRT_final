# miniRT 프로그램 개요

## 프로젝트 소개

miniRT는 C 언어로 구현된 레이 트레이싱(Ray Tracing) 렌더러입니다. 3D 장면을 2D 이미지로 변환하는 기본적인 렌더링 기법을 학습하고 구현하는 것을 목표로 합니다.

## 주요 기능

### 1. 지원하는 기하학적 객체
- **구(Sphere)**: 중심점과 반지름으로 정의
- **평면(Plane)**: 한 점과 법선 벡터로 정의되는 무한 평면
- **원기둥(Cylinder)**: 중심, 축 방향, 반지름, 높이로 정의되는 유한 원기둥 (캡 포함)

### 2. 조명 시스템
- **Ambient Light(주변광)**: 모든 객체에 균일하게 적용되는 기본 조명
- **Point Light(점광원)**: 특정 위치에서 모든 방향으로 빛을 발산
- **Phong Lighting Model**: 
  - Ambient: 기본 조명
  - Diffuse: 난반사광 (표면의 거친 반사)
  - Specular: 정반사광 (반짝이는 하이라이트)

### 3. 그림자 시스템
- **Soft Shadows(부드러운 그림자)**: 
  - 다중 샘플링 기법 사용 (기본 16개 샘플)
  - 그림자 경계의 자연스러운 블러 효과
  - Penumbra(반음영) 구현
- **Shadow Acne 방지**: 적응형 바이어스 사용

### 4. 카메라 시스템
- 위치, 방향, 시야각(FOV) 설정 가능
- 원근 투영(Perspective Projection)

## 기술 스택

- **언어**: C (C99 표준)
- **그래픽 라이브러리**: MinilibX
- **빌드 시스템**: Makefile
- **수학 라이브러리**: 자체 구현 (vec3 연산)

## 프로젝트 구조

```
miniRT/
├── includes/          # 헤더 파일
│   ├── minirt.h      # 메인 헤더
│   ├── objects.h     # 객체 정의
│   ├── ray.h         # 레이 구조체
│   ├── vec3.h        # 벡터 연산
│   ├── shadow.h      # 그림자 시스템
│   ├── parser.h      # 파싱 관련
│   └── window.h      # 윈도우 관리
├── src/              # 소스 파일
│   ├── main.c        # 프로그램 진입점
│   ├── parser/       # 씬 파일 파싱
│   ├── ray/          # 레이-객체 교차 검사
│   ├── render/       # 렌더링 로직
│   ├── lighting/     # 조명 및 그림자 계산
│   ├── math/         # 수학 연산 (벡터)
│   └── window/       # 윈도우 및 이벤트 처리
├── scenes/           # 씬 파일 (.rt)
├── docs/             # 문서
└── Makefile          # 빌드 설정
```

## 입출력

### 입력
- `.rt` 확장자를 가진 씬 파일
- 텍스트 형식으로 장면 설명

### 출력
- X11 윈도우에 렌더링된 이미지 표시
- ESC 키 또는 창 닫기로 종료

## 실행 방법

```bash
# 빌드
make

# 실행
./miniRT scenes/cylinder_test.rt

# 정리
make clean      # 오브젝트 파일 삭제
make fclean     # 실행 파일까지 삭제
make re         # 재빌드
```

## 성능 특성

- **렌더링 방식**: CPU 기반 레이 트레이싱
- **해상도**: 800x600 (기본값)
- **렌더링 시간**: 복잡도에 따라 수초에서 수십초
- **메모리 사용**: 씬 데이터와 프레임버퍼
