## 📋 문서 작성 규칙

### Constitution v1.2.1 준수
본 문서 구조는 [42 miniRT Constitution v1.2.1](../.specify/memory/constitution.md)의 Principle VII을 따릅니다.

### 파일 명명 규칙
- 모든 파일명은 한글과 언더스코어(_) 사용: `파일_이름.md`
- 구현 요약은 이슈 번호 접두사 사용: `NNN_기능_설명.md`
- 날짜 기반 파일은 YYYY-MM-DD 접두사 사용 (필요시)

### 상호 참조
- 문서 간 링크는 상대 경로 사용
- 예: `[파일 구조](../02-개발/파일_구조.md)`

### 문서 생명주기
- 새 기능 문서 → `03-기능/[하위시스템]/`
- 구현 완료 요약 → `05-프로젝트_히스토리/구현_요약/`
- 오래된 문서 → `06-참고자료/` (날짜 접두사 추가)
