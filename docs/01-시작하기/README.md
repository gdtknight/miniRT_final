# 01-시작하기

miniRT 프로젝트를 시작하는 데 필요한 **실용적 가이드와 사용법**을 제공합니다.

> 💡 **알고리즘이 궁금하신가요?** 기술 문서는 [00-전체설명/](../00-전체설명/)을 참조하세요.

## 📄 문서 목록

### [miniRT_요구사항.md](./miniRT_요구사항.md) ⭐ 필수
42 school miniRT 프로젝트의 공식 요구사항과 명세서입니다.
- ✅ 필수 기능 목록 및 구현 요구사항
- 📋 평가 기준 및 체크포인트
- ⚠️ 제약 조건 (허용/금지 함수, 라이브러리)
- 🎓 보너스 기능 (선택사항)

### [씬_파일_가이드.md](./씬_파일_가이드.md) 📝 실습
miniRT의 씬 파일(.rt) 형식과 작성 방법을 설명합니다.
- 📐 씬 파일 문법 및 규칙
- 🔵 오브젝트 정의 (sphere, plane, cylinder)
- 💡 조명 설정 (ambient, light)
- 📷 카메라 구성 및 FOV
- 🎨 색상 및 재질 설정
- ✏️ 실전 예제 및 템플릿

## 🚀 빠른 시작

### 1단계: 요구사항 확인
```bash
# miniRT_요구사항.md를 읽고 과제 범위를 파악하세요
```

### 2단계: 프로젝트 빌드
```bash
make          # miniRT 컴파일
make bonus    # 보너스 기능 포함
```

### 3단계: 첫 씬 실행
```bash
./miniRT scenes/simple.rt
```

### 4단계: 씬 파일 작성
`씬_파일_가이드.md`를 참고하여 자신만의 씬을 만들어보세요!

## 🎯 대상 독자

- ✨ miniRT 프로젝트를 처음 시작하는 개발자
- 📝 씬 파일을 작성하려는 사용자  
- 📋 프로젝트 요구사항을 확인하려는 평가자/동료
- 🔍 과제 스펙을 검증하려는 리뷰어
