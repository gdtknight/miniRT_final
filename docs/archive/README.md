## 🗃️ 아카이브 정책

### 아카이브 대상
1. **중복 문서**: 내용이 중복되거나 통합된 문서
2. **완료된 일회성 보고서**: 특정 시점의 상태를 기록한 보고서
3. **레거시 계획**: 실행 완료되었거나 대체된 계획 문서
4. **백업 파일**: 마이그레이션 시 생성된 백업

### 아카이브 절차
1. 문서를 `archive/YYYY-MM-legacy/` 디렉토리로 이동
2. 원본 위치에 리다이렉트 메모 남기기 (필요시)
3. 이 README에 항목 추가

### 보존 기간
- **최소 1년**: 모든 아카이브 문서
- **영구 보존**: 주요 이정표 관련 문서
- **정기 검토**: 매년 12월
